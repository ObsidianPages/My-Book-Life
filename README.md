# My Book Life

An aesthetic, interactive reading tracker built with:

- Next.js (App Router)
- TailwindCSS
- Supabase (Auth + Database)

## Features

- Book tracking
- Author files
- Book boyfriend files
- Favourite quotes
- Book aesthetics moodboards
- Reading goals
- Dashboard with stats

## Setup

1. Create a Supabase project
2. Add the SQL schema from `/schema.sql`
3. Copy `.env.local.example` → `.env.local`
4. Add your Supabase URL + anon key
5. Deploy to Vercel
